# RPG Character Creator

A Python-based RPG character creation system that validates character attributes and generates a character sheet.

## Features

- Create characters with customizable names and stats
- Input validation for:
  - Character names (no spaces, appropriate length)
  - Stat values (must be integers within valid range)
- Simple character sheet display showing current stats

## Character Attributes

Each character has the following attributes:
- **Name**: A unique identifier for your character
- **Strength**: Physical power and combat ability
- **Intelligence**: Mental acuity and magical prowess
- **Charisma**: Social influence and leadership
